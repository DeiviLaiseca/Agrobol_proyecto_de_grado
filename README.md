
# Agrobol OEE Prediction App

Esta aplicación permite cargar datos manualmente o mediante archivo CSV para predecir el OEE utilizando un modelo Gradient Boosting entrenado previamente.

## Funcionalidades:
- Predicción del OEE.
- Opción de carga manual o por archivo.
- Validación automática de columnas requeridas.
- Visualización clara y limpia con paleta de colores Agrobol.

## Tecnologías:
- Python
- Streamlit
- Scikit-learn
- Joblib
